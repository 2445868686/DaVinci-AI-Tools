DaVinci Resolve Image AI 安装指南

步骤 1：复制脚本文件

将 DaVinci Image AI 文件夹放入 Resolve 脚本目录。如果本文件与脚本已在同一目录，可跳过。

Windows 路径

C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Utility

macOS 路径

~/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Utility

步骤 2：在 Resolve 中启用

  • 启动或切回 DaVinci Resolve。
  • 依次选择 工作区 → 脚本 → DaVinci Image AI。

DaVinci Resolve Image AI 使用指南

功能简介

集成 Google Gemini 与 Seed Dream，支持文生图、图生图、从时间线提取帧后编辑，结果可预览、保存并加入媒体池。中英双语 UI，配置好 API Key 即可使用。

界面结构

  • 图片生成：服务商、画幅/清晰度、提示词、生成按钮、结果预览。
  • 图片编辑：原图/结果预览、提示词、服务商配置、生成按钮、状态栏。
  • 设置：保存路径、Google/Seed Dream 配置、语言切换。
  • 状态栏：显示校验、请求、保存等提示。

文生图 / 图生图

  • 在“图片生成”页输入提示词，选择服务商（Google Gemini/Pro 或 Seed Dream），画幅、清晰度后点击 生成。
  • 生成结果会显示在预览区并保存到设置的目录，可点击 添加到媒体池 导入媒体池。
  • 图生图：先上传图片或 提取当前帧，再输入提示词生成新图。

图片编辑

  • 在“图片编辑”页点击 选择图片 或 提取当前帧 选取原图。
  • 描述想要的修改，选择服务商后点击 开始生成，编辑结果会显示在预览区。
  • 可用“继续编辑”将结果设为新原图迭代调整；保存路径按设置页为准。

设置

  • 保存路径：指定生成/编辑后图片的存放位置。
  • Google 配置：Base URL（默认 https://generativelanguage.googleapis.com）与 API Key。
  • Seed Dream 配置：服务地址、API Key、默认模型。
  • 语言：切换中/英文界面。

常见问题

预览有图但提示“请先选择一张图片”？

生成按钮检查的是源文件路径，若原文件被清除/删除需重新上传或提取帧。

提取时间线帧尺寸过大？

脚本提取后会尝试压缩预览，仍嫌大可在导出后再下采样，或降低时间线分辨率后再提取。

API 报错或无响应？

检查 Base URL、API Key 及网络/代理配置，稍后重试。

DaVinci Resolve Image AI Installation Guide

Step 1: Copy the Script

Place the DaVinci Image AI folder in the Resolve script directory. If this guide sits beside the script already, skip the copy.

Windows Path

C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Utility

macOS Path

~/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Utility

Step 2: Enable inside Resolve

  • Launch DaVinci Resolve.
  • Go to Workspace → Scripts → DaVinci Image AI.

DaVinci Resolve Image AI Usage Guide

Overview

Integrates Google Gemini and Seed Dream for text-to-image and image-to-image workflows. Generate from prompts, edit existing images (including timeline frames), preview results, save, and add to the media pool. Chinese/English UI included—configure API keys and go.

Interface Layout

  • Generate: Provider selection, aspect/quality, prompt input, generate button, result preview.
  • Edit: Original/result previews, prompt input, provider config, generate button, status line.
  • Settings: Save path, Google/Seed Dream configs, language toggle.
  • Status: Shows validation, request, and save messages.

Text-to-Image / Image-to-Image

  • On the “Generate” tab, enter a prompt, choose provider (Google Gemini/Pro or Seed Dream), aspect, and quality, then click Generate.
  • Results preview in-place and save to your configured folder; use Add to Media Pool to import to the pool.
  • For image-to-image, upload an image or Grab Current Frame, then generate with your prompt.

Image Edit

  • On “Edit”, click Pick Image or Grab Current Frame to set a source.
  • Describe the change, choose provider, and click Generate; the edited image appears in the result preview.
  • “Continue Editing” sets the result as the new source for iterative refinement; saved files follow your configured path.

Settings

  • Save Path: Where generated/edited images are stored.
  • Google Config: Base URL (default https://generativelanguage.googleapis.com) and API Key.
  • Seed Dream Config: Service URL, API Key, and default model.
  • Language: Switch UI between Chinese and English.

FAQ

Preview shows an image but “Please pick an image first” appears.

The button checks the source path; if the file was cleared/deleted, re-upload or grab the frame again.

Timeline frame export is too large.

The script compresses the frame for preview (macOS uses sips). For smaller files, downscale further after export or lower timeline resolution before grabbing.

API errors or no response.

Verify Base URL and API Key, check proxy/network settings, and retry.
