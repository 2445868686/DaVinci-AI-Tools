DaVinci Resolve Sub Translator 安装指南

步骤 1：检查 DaVinci Resolve 中是否已安装 Python

点击：菜单栏——>工作区 ——> 控制台 ——> Py3。如果已安装，则不会弹出窗口（直接进入 步骤 2：安装插件）。

若未安装，则会弹出 "Python was not found" 消息，请根据提示下载并安装，安装完成后进入步骤 2：安装插件！

⚠️ Windows 用户：安装 Python 时请务必勾选 "Add Python to PATH" 选项，否则可能需要重新安装或手动设置环境变量。

步骤 2：安装插件

Mac 用户：

双击 Mac_Install.command进行安装，等待安装完成。

Win 用户：

双击 Windows_Install.bat进行安装，等待安装完成。

安装完成后，可通过 菜单栏——>工作区 ——> 脚本 打开插件进行使用。

DaVinci I2V 使用指南

DaVinci I2V 是一个“图生视频”插件：给出首帧（可选尾帧）与提示词，由在线模型生成短视频，并自动导入到达芬奇媒体池与时间线中。

一、 主界面功能（点击展开）

  • 页签：海螺视频（MiniMax）、Runway 与 配置。
  • 生成模式：顶部会显示“当前模式”；当模型支持“首尾帧”时，标题会自动切换为“首尾帧”模式。
  • 选择首帧：点击“选择首帧”从本地挑选 PNG/JPG（≤10MB，建议≥300px）。
  • 选择尾帧：当模型支持时可用；某些模型或分辨率下会自动禁用。
  • 交换按钮：点击“⇆”可将首帧与尾帧图片互换。
  • 支持性：
    • MiniMax：MiniMax-Hailuo-02 在 768P/1080P 支持尾帧；其它 I2V-01 系列不支持尾帧。
    • Runway：gen3a_turbo 支持尾帧，gen4_turbo 仅首帧。
  • 在文本框中输入生成提示词，影响画面风格与内容。可留空。
  • MiniMax 模型：MiniMax-Hailuo-02、I2V-01、I2V-01-live、I2V-01-Director。
    • 时长：MiniMax-Hailuo-02 支持 6s/10s；其余固定 6s。
    • 分辨率：MiniMax-Hailuo-02 在 6s 时 512P/768P/1080P，10s 时 512P/768P；其他模型 720P。
  • Runway 模型：gen4_turbo、gen3a_turbo。
    • 分辨率层级：gen4_turbo 为 720P，gen3a_turbo 为 768P；横竖屏比例将依据首帧图片自动决定。
    • 时长：5s/10s 可选。
  • 点击“生成”会提交任务并自动轮询进度；完成后自动下载并导入时间线。
  • 如已获得 任务ID，可粘贴到“任务ID”输入框，点击“下载”仅进行结果查询与下载。
  • 保存路径：在“配置”页点“浏览”选择目录，插件会自动创建 项目名_I2V 子文件夹并写入文件。
  • API 设置：点击 Minimax/Runway 的“配置”按钮填写 Base URL 与 API Key。
  • 界面右下提供 EN/简体中文 复选框；也可使用本页面右上角语言切换查看文档。

二、 快速上手

  1. 在“配置”页选择 保存路径，并设置 API Key。
  2. 切换到 海螺视频（MiniMax） 或 Runway 页。
  3. 点击 选择首帧（必要），若模型支持可再选 尾帧。
  4. （可选）输入 提示词，并根据需要选择 模型/时长/分辨率。
  5. 确认时间线处于目标位置，点击 生成，等待完成自动入库并添加到时间线空轨道。

三、 注意事项

  • 本插件调用在线服务，需确保 网络可用 且账号 余额充足。
  • 不同模型对 尾帧 支持不同；当不支持时按钮会自动禁用。
  • 下载完成后，视频会导入媒体池并添加到 第一个空的视频轨道，从 当前指针 到时间线末尾。
  • 若遇到 API Key、限频、余额不足等错误，插件会在弹窗中给出提示。

DaVinci I2V – Installation Guide

Step 1: Check for Python in DaVinci Resolve

In Resolve, go to Menu Bar → Workspace → Console → Py3. If the Python console opens, proceed to Step 2: Install Plugin. If you see “Python was not found”, install Python first.

Windows: during installation, make sure to select Add Python to PATH.

Step 2: Install Plugin

macOS:

Double-click Mac_Install.command to install and wait for completion.

Windows:

Double-click Windows_Install.bat to install and wait for completion.

DaVinci I2V – Usage Guide

DaVinci I2V generates video from images using online models. Provide a first frame (and optionally a last frame) plus a prompt, then the clip is generated, downloaded and inserted onto your timeline automatically.

I. UI Overview (click to expand)

  • Tabs: Hailuo (MiniMax), Runway, and Configuration.
  • Mode: When a model supports first & last frames, the banner shows “First & Last Frame”. Otherwise it’s “Image to Video”.
  • First Frame: choose a local PNG/JPG (≤10MB, ≥300px recommended).
  • Last Frame: available only when the chosen model/resolution supports it.
  • Swap: click “⇆” to swap the two images.
  • Support:
    • MiniMax: MiniMax-Hailuo-02 supports last frame at 768P/1080P; I2V‑01 series do not.
    • Runway: gen3a_turbo supports last frame; gen4_turbo requires first frame only.
  • Enter a text prompt to guide content and style. Optional.
  • MiniMax Models: MiniMax-Hailuo-02, I2V-01, I2V-01-live, I2V-01-Director.
    • Duration: MiniMax-Hailuo-02 offers 6s/10s; others fixed 6s.
    • Resolution: For MiniMax-Hailuo-02 → 512P/768P/1080P at 6s, and 512P/768P at 10s; other models use 720P.
  • Runway Models: gen4_turbo, gen3a_turbo.
    • Resolution tier: gen4_turbo uses 720P, gen3a_turbo uses 768P. The aspect ratio is chosen automatically from your first image (landscape/portrait).
    • Duration: 5s/10s.
  • “Create” submits a task and polls its status. When finished, the video is downloaded and placed onto the timeline.
  • If you already have a Task ID, paste it and click “Download” to fetch the result directly.
  • Save Path: choose a folder in “Configuration.” The plugin creates a <ProjectName>_I2V subfolder there.
  • API: click “Config” next to Minimax/Runway to set Base URL and API Key.
  • Use the in‑plugin EN/中文 toggles, or the buttons on this page to read the docs.

II. Quick Start

  1. Set Save Path and API Keys in Configuration.
  2. Switch to Hailuo (MiniMax) or Runway.
  3. Pick a First Frame (required). If supported, also pick a Last Frame.
  4. Optionally enter a Prompt and set Model/Duration/Resolution.
  5. Position the playhead and click Create. The clip is inserted on the first empty video track from the playhead to the end.

III. Notes

  • Online services are required. Ensure network access and sufficient balance.
  • Last‑frame support depends on model/resolution; the control is disabled when unsupported.
  • After download, the clip is imported and placed on the first empty video track from the current playhead.
  • API errors (key/limits/balance) are shown via in‑plugin messages.
